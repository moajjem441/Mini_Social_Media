# Mini Social Media App (java Swing)

A simple desktop social media app which made by Java Swing and OPP concept


# key Features
.Authentication:user registration and log in
.Post & Interact :it has benebits of text posting,liking and commenting.
.Dark Mode: User can change the mood by clicking the setting button.
.Data Storage :Sava the local data by using Java Serialization.
.Profile:Dashboard show the total post and likes


# Tech Stack
.Language: Java
.UI: Java Swing & AWT
.Storage : File I/O(.dat files)
